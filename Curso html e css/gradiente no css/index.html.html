Mudando de cor de fundo

Para gradientes com mais cores, apenas adicionar a cor desejada após a ultima cor separando as com virgula!

  • Para ir da direita para a esquerda usar usar no css a propriedade “background-image: linear-gradient (to left, white, blue);”

  • Para ir de baixo para cima usar usar no css a propriedade “background-image: linear-gradient (to top, white, blue);”

  • Para o gradiente ir da esquerda para a direita usar no css a propriedade “background-image: linear-gradient (to right, white, blue);”

  • Para ir de cima para baixo usar usar no css a propriedade “background-image: linear-gradient (to bottom, white, blue);”
