Três formas de se aplicar o Css

A primeira forma é pelo link externo, ou seja, um documento CSS externo com um link no documento html:

Para usar um link externo no html, utliza-se o código <link rel="stylesheet" href="style.css"> na tag head do documento, onde o o atributo href do código recebe o nome do documento.css


A segunda forma é bem mais simples, apenas adicionando na tag head um elemento filho chamado <style> com fechamento e dentro dessa tag adicionar o estilo css. Ex.
<head>
......<style>

......</style>
</head>

A terceira forma é importando o estilo CSS de um documento, para isso deve ser escrito "@ import url (pasta do arquivo/arquivo)"
